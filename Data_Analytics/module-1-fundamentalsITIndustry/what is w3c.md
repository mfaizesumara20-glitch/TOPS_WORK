# what is w3c ?

1. W3C stands for World Wide Web Consortium.
2. It is an international community that develops open standards to ensure the long-term growth of the Web. The W3C was founded in 1994 by Tim Berners-Lee, the inventor of the World Wide Web.
3. The W3C is responsible for creating and maintaining web standards such as HTML, CSS

4. JavaScript, and many others. These standards ensure that web content is accessible, interoperable, and works consistently across different browsers and devices.
4. The W3C also promotes the development of web technologies and provides guidelines for web developers
