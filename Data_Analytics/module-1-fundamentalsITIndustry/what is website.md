# what is website ?

 1. A website is a collection of related web pages that are typically identified by a common domain name and published on at least one web server. Websites can be accessed via the internet using a web browser.

 2. A website is a collection of information in form of text, audio , videos. home pages , web pages  that is called website.


 # types of website

 1) dynamic website : A dynamic website is a type of website that generates content in real-time based on user interactions or other factors. It typically uses server-side scripting languages like PHP, Python, or JavaScript to create and display content dynamically. Examples of dynamic websites include social media platforms, e-commerce sites, and news websites.

2) static website : A static website is a type of website that consists of fixed web pages that are delivered to the user's browser exactly as they are stored on the server. Static websites do not require any server-side processing and are typically created using HTML, CSS, and JavaScript. Examples of static websites include personal blogs, portfolios, and informational websites.
