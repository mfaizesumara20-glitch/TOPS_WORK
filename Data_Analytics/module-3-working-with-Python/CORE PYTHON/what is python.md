## What is an Operator in Python?

An **operator** is a symbol that performs an operation on one or more values (called **operands**).

### Syntax

```python
operand operator operand
```

### Example

```python
10 + 5
```

Here:
- `10` and `5` are operands
- `+` is the operator

---

# Types of Operators in Python

## 1. Arithmetic Operators

Used to perform mathematical calculations.

### Examples

```python
+   # Addition
-   # Subtraction
*   # Multiplication
/   # Division
%   # Modulus
//  # Floor Division
**  # Exponent
```

---

## 2. Comparison Operators

Used to compare two values. The result is either `True` or `False`.

### Examples

```python
>   # Greater than
>=  # Greater than or equal to
<   # Less than
<=  # Less than or equal to
==  # Equal to
!=  # Not equal to
```

**Note:** Python uses `==`, not `===`.

---

## 3. Assignment Operators

Used to assign values to variables.

### Examples

```python
=    # Assignment
+=   # Add and assign
-=   # Subtract and assign
*=   # Multiply and assign
/=   # Divide and assign
%=   # Modulus and assign
```

### Example

```python
x = 10
x += 5   # x becomes 15
```

---

## 4. Logical Operators

Used to combine conditions.

### Examples

```python
and
or
not
```

### Example

```python
age = 20
salary = 30000

print(age > 18 and salary > 25000)
```

---

## 5. Conditional (Ternary) Operator

Python does not have the traditional `?:` ternary operator, but it provides a conditional expression.

### Syntax

```python
value_if_true if condition else value_if_false
```

### Example

```python
age = 18

result = "Adult" if age >= 18 else "Minor"
```

---

## 6. Bitwise Operators

Used to perform operations on binary numbers.

### Examples

```python
&   # AND
|   # OR
^   # XOR
~   # NOT
<<  # Left Shift
>>  # Right Shift
```

**Note:** `+=`, `-=`, `*=`, `/=` are assignment operators, not bitwise operators.

---

# Variables in Python

## What is a Variable?

A **variable** is a name used to store data in memory.

### Example

```python
name = "Faize"
age = 20
salary = 25000
```

Here:
- `name`, `age`, and `salary` are variables.
- `"Faize"`, `20`, and `25000` are values stored in those variables.
