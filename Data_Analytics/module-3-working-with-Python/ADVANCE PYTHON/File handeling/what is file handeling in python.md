## file handling modes

- r: read only
- r+: read and write only
- w: write only but if file does not exist it creates a new file
- w+: read | wrrite but if file is not available it creates new file
- a: append on file if file does not exist it creates a new file
- a+: append and read on file if file does not exist it creates a new file
